# Predict Data Analytics System

## 📌 Project Overview

Predict Data Analytics System is a Machine Learning and Data Analytics project developed during my internship at ReadyNest Corp.

The project analyzes Flipkart product reviews and predicts customer ratings using a Logistic Regression model. It also provides interactive visualizations and business insights through a Streamlit dashboard.

---

## 🚀 Features

- Dataset Overview
- Data Cleaning
- Interactive Visualizations
- Rating Prediction
- Business Insights
- Streamlit Dashboard

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Plotly
- Scikit-learn
- Streamlit
